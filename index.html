スキル

HTML

HTMLタグで１からウェブサイトの構築ができます。アルバイトでHTMLを使ってWebサイト制作した経験があります。

CSS

作成されたHTMLサイトに対し、CSSを使って、デザインを加えることができます。keyframeのアニメーションやRWD対応ができます。 BootstrapとTailwind CSSなどのフレームワークも独学で制作物にて使ってました。最近はScssというCSSのメタ言語を勉強しております。

JAVASCRIPT

Node.jsでDiscordのチャットボットを作ることで、Javascriptの勉強を始めました。Javascriptで動的にウェブサイトのコンテンツを更新することができます。またjQueryのプラグインを実装することができます。 さらにReact.jsなどのフレームワークでシンプルなウェブアプリを開発したことがあります。 Vue.jsも勉強しております。

Adobe XD

XDを使ってUIデザインすることができます。

ILLUSTRATOR

アイコンの作成、編集やWebサイトのデザインができます。 例えば、制作物の中のWebサイトやアプリのロゴセットをデザインし、開発へ進みました。

PHOTOSHOP

写真の色味調整、背景抜き及び基本の加工ができます。大学の卒業制作にてPhotoshopを使って自分の雑誌など作ったことがあります。アルバイトでもPhotoshopで施策の必要に応じて画像を修正・制作した経験があります。

JAPANESE

2017年にJLPT N1に合格しました。
今まで約二年弱日系企業で働き、通常業務に関する会話から基本のビジネスメールができます。現在専門学校へ入学、Webの勉強をしており、Web制作会社にてアルバイトをさせて頂いております。

DIGITAL PHOTOGRAPHY

デジタルカメラで人物の撮影が得意で、大学卒業制作に撮影の担当をさせていただきました。制作物で使ったお写真全て自分で撮影しました。さらに、Lightroomというツールで写真の加工ができます。

制作物

Weekly Menu!

HAL EVENT WEEKという進級制作展に向けて、なるべく多くの人々に使って頂きたくてメニューを決めてくれるアプリを制作しました。食事を決めるのに時間がかかってしまう方は、このアプリが一週間のリストをランダムで決めてくれます。さらにRecipe APIの導入により、手入力リストインの手間が省けます。
開発環境：React.js / Firebase realtime database / Rakuten Recipe APIs
受賞歴：HAL EVENT WEEK銅賞　入賞

サイトへ

NudoNudo

仮想のカメラマンサービスサイトです。自分自身の趣味である人物撮影が好きで、学校の課題として、カメラマンサービスを提供している仮想の公式サイトを企画してみました。技術的に苦労したところはScss使ったのは初めてで理解に時間かかったところと、fullpage.jsというjQueryを使った原因でRWD的にCSSの調整が大変でした。

サイトへ

CHONGYUE

台湾にある実家のお店の公式サイトを想定して制作しました。暗めの配色を基調に、20代〜50代まで男性をメインにターゲットにしています。

サイトへ

Gallery Album

趣味で撮影した写真をギャラリーにしようと考えて、UIデザインからコーディングまで自分で出来ました。UIデザインはAdobe Illustratorで、開発環境はReact.js + Gatsby.js + CSSでございます。これからはRWSの実装とフレーム化して一つのオープンソースにしたいと思います。

サイトへ

My Resume

独学で勉強したHTMLとCSSスキルを使って自分らしい紹介ページを作りました。各種レイアウト、配置からアニメション、RWD（レスポンシブウェブデザイン）までの実践でございます。自分で作ったアイコンや写真などは載せておりますので、個性も伝わるかと思っております。是非、ご覧いただければ幸いでございます。

ガチャ確率計算機

ソーシャルゲーム用のガチャ計算アプリです。簡単なScriptに、自分で考えたレイアウトを載せました。自分にとって、これもCSSに触れてから初めて「自分が制作した」と言える物です。OBSなどのストリームツールで、ガチャの詳細を実況中に視聴者たちに分かりやすくお見せできるという概念で作りました。バックグラウンドとへーダーのカスタマイズ機能も搭載しました。

CSS Animation

勉強したCSSだけで作ってみたWebアニメーションでございます。@keyframeとアニメーションの構成についての練習です。ちょうど自分が撮影した「最高の一枚」もそこでお披露目することができました。
